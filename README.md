# 🤖 Qwen3.5-2B 金融智能投研助手

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> **本地化运行的金融数据专家** | 基于 Qwen3.5-2B 大模型与 AkShare 实时数据

本项目构建了一个轻量级的本地金融分析助手。它利用 **Qwen3.5-2B** 大模型进行自然语言理解，并结合 **AkShare** 专业金融数据接口，实现了对 A股行情、历史走势及宏观经济数据的精准查询。有效解决了通用大模型在金融数据上的“幻觉”与“数据滞后”问题。

---

## ✨ 核心功能

- **实时行情查询** 📈
  - 支持通过股票名称（如“贵州茅台”）或代码（如“600519”）查询实时价格、涨跌幅、成交量。
- **历史走势分析** 📊
  - 自动获取指定时间段的日K线数据，并计算最高价、最低价及平均收盘价。
- **宏观数据洞察** 🏦
  - 实时获取中国年度 GDP 与 CPI 数据，辅助判断经济周期。
- **本地化部署** 💻
  - 支持 CUDA (GPU) 与 CPU 模式自动切换，低资源环境也可运行。

## 🚀 快速开始

### 1. 环境准备
推荐使用 Python 3.9+ 环境。

```bash
# 克隆项目
git clone https://github.com/yourname/finance-agent.git
cd finance-agent
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 启动应用
```bash
python main.py
```

**交互示例：**
```text
👤 用户: 贵州茅台现在的股价是多少？
🤖 助手: ✅ **实时行情查询结果** ...
```

## 🛠️ 技术架构

- **LLM 模型**: `Qwen/Qwen3.5-2B` (通过 ModelScope 加载)
- **数据源**: [AkShare](https://github.com/akfamily/akshare) (东方财富、新浪财经等)
- **核心逻辑**: 
  - **意图识别**: 基于关键词的优先级路由（宏观数 -> 实时 -> 历史）。
  - **数据清洗**: 动态列名匹配，防止接口更新导致报错。
  - **设备管理**: 自动检测 CUDA 环境，支持 `float16` 推理加速。
